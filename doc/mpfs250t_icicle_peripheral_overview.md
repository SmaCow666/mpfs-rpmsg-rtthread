# MPFS250T-Icicle-Kit 外设资源清单

## 文档说明
1. 本文档基于Microchip官方技术手册、板卡原理图整理，分为**MSS内置硬化外设**、**板级外围器件与扩展接口**两大板块；
2. 适配硬件平台：PolarFire SoC Icicle Kit（主控：MPFS250T）。

---

# 一、MSS 内置硬化外设
## 1.1 处理器核心
| 名称 | 参数说明 |
| ---- | -------- |
| RISC-V多核子系统 | 1颗E51监控核（RV64IMAC）+4颗U54应用核（RV64GC），U54核支持Sv39 MMU内存管理单元，芯片总计5个硬件Hart |

## 1.2 存储控制器
| 名称 | 参数说明 |
| ---- | -------- |
| DDR控制器 | 兼容DDR3/DDR3L/DDR4/LPDDR3/LPDDR4，板载LPDDR4内存直接挂载至该控制器 |
| QSPI-XIP控制器 | 1路控制器，支持Flash就地执行（XIP），对接板载1Gb SPI Flash |
| eMMC/SD/SDIO控制器 | 1路三合一控制器，支持eMMC5.1、SD、SDIO协议，对接板载eMMC颗粒与SD卡座 |
| eNVM控制器 | 嵌入式非易失性存储器控制器，芯片原生启动代码存储专用硬件外设 |

## 1.3 通信外设
| 名称 | 参数说明 |
| ----- | ------------ |
| MMUART | 共5路串口（MMUART0~MMUART4），引脚分配至MSS Bank2、Bank4 |
| SPI控制器 | 2路SPI接口，仅支持主机模式，最高通信时钟为系统半主频HCLK/2 |
| I2C控制器 | 2路I2C接口，支持主/从机模式，标准速率100Kbps、高速速率400Kbps |
| CAN控制器 | 2路CAN控制器，内置32级发送缓冲区、32级接收缓冲区 |
| 千兆以太网MAC（GEM） | 2路原生千兆MAC，支持SGMII接口，兼容10/100/1000Mbps自适应速率 |
| USB2.0 OTG控制器 | 1路USB2.0 OTG控制器，外接ULPI物理层接口 |
| GPIO模块 | 分为3个GPIO硬件块，总计96路通用IO，板载用户LED、按键均挂载至该模块 |

## 1.4 定时与中断外设
| 名称 | 参数说明 |
| ---- | -------- |
| 32位通用定时器 | 内置2路独立32位硬件定时器 |
| 硬件看门狗 | 内置5路独立看门狗定时器，用于系统异常复位防护 |
| RTC实时时钟 | 1路硬件实时时钟，支持断电走时 |
| FRQ频率计 | 1路硬件频率测量外设，用于外部时钟信号频率采集 |
| CLINT核本地中断控制器 | RISC-V标准本地中断控制器，提供内核mtime定时器、核间软件中断能力 |
| PLIC平台级中断控制器 | 全局中断管理控制器，统一管理芯片169路全局硬件中断 |

## 1.5 其他内置外设
| 名称 | 参数说明 |
| ---- | -------- |
| 硬件加密处理器 | 内置Athena F5200 TeraFire硬件加密引擎，支持国密及通用加解密算法 |
| FIC Fabric接口控制器 | 共计5路总线接口：3路64位AXI4、1路32位APB、1路32位AHB-Lite，实现MSS与FPGA Fabric高速数据交互 |
| M2F中断控制器 | 专用跨域中断控制器，负责MSS处理器子系统与FPGA逻辑侧之间的中断路由与转发 |

# 二、板级外围器件与扩展接口
## 2.1 存储类器件
| 名称 | 参数规格 | 挂载位置 |
| ---- | -------- | -------- |
| LPDDR4内存 | MT53D512M32D2DS-053，容量16Gb，工作频率800MHz | MSS Bank6 |
| SPI Flash | MT25QL01GBBB8ESF-0SIT，容量1Gb，用于固件存储 | MSS Bank3 |
| eMMC固态存储 | SDINBDG4-8G，容量8GB，兼容eMMC5.1协议 | MSS eMMC控制器 |
| Micro SD卡座 | 支持UHS-I高速SD卡协议，外置存储卡扩展 | MSS SD控制器 |

## 2.2 板载通信接口
| 名称 | 参数规格 | 挂载位置 |
| ---- | -------- | -------- |
| 双路千兆以太网 | VSC8662双口千兆PHY，2路RJ45接口，10/100/1000M自适应 | MSS GEM千兆MAC |
| USB2.0 OTG接口 | USB3340 USB PHY，Micro-AB接口，支持主从双向模式 | MSS USB OTG控制器 |
| 四路USB转串口 | CP2108桥接芯片，单USB扩展4路硬件串口 | FPGA Bank1 |
| 双路CAN总线接口 | MCP2562FDT-E/SN CAN收发器，2组3pin工业接线端子 | MSS CAN控制器 |
| PCIe x16接口 | 4路SerDes差分通道，标准PCIe金手指插槽 | 芯片原生SerDes硬核 |

## 2.3 通用扩展接口
| 名称 | 参数规格 | 挂载位置 |
| ---- | -------- | -------- |
| mikroBUS扩展接口 | 2组16pin标准座子，引出UART/SPI/I2C外设 | FPGA IO |
| 树莓派40pin扩展座 | 兼容树莓派标准引脚定义，40pin排针接口 | GPIO Bank1、GPIO Bank9 |

## 2.4 调试与编程接口
| 名称 | 参数说明 |
| ---- | -------- |
| 板载FlashPro6编程器 | 板载一体化下载调试器，USB直连PC即可完成固件烧录与在线调试 |
| 外置JTAG调试座 | 10pin标准JTAG接口，可外接FlashPro4/5/6系列离线编程器 |

## 2.5 板载时钟源
| 名称 | 参数规格 | 挂载位置 |
| ---- | -------- | -------- |
| MSS参考时钟 | 125MHz LVDS差分晶振 | MSS处理器子系统 |
| FPGA系统时钟 | 50MHz单端有源晶振 | FPGA可编程逻辑阵列 |

## 2.6 用户交互硬件
| 名称 | 参数规格 | 挂载位置 |
| ---- | -------- | -------- |
| 用户LED指示灯 | 4路可编程LED，低电平点亮 | HSIO Bank0 |
| 用户功能按键 | 4路独立轻触按键，按下输出低电平 | HSIO Bank0 |

## 2.7 电源与复位电路
| 名称 | 参数说明 |
| ---- | -------- |
| 电源电流监控芯片 | PAC1934T-I/JQ，I2C接口，实时监测4路核心电源电流 |
| 电源状态指示灯 | 9路电源检测LED，全覆盖板卡所有电压轨供电状态 |
| 电源输入与总开关 | 12V/5A DC电源输入接口，搭配硬件电源总开关 |
| 系统专用复位芯片 | MCP121T-315E，统一生成全局系统复位信号DEVRST_N |
| 板载电源稳压芯片 | 集成MIC26950、MIC22705等多路电源芯片，为全板硬件提供额定供电 |
